Wow! This is served from a Node server

Incredible!

This is an ultra basic app that uses vanilla Node — no Express.js — to return an HTML file.

The HTML file also makes requests for CSS and JS files, so the Node server has to handle those too.

It checks for the request's extension first. If it includes ".css", it returns the only stylesheet available — "style.css".

It does the exact same thing for JavaScript.

This feels highly inefficient, but it's a start.

I highly doubt I will update this page again, so if you stumble upon it — hello! Node is cool!
